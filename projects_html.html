Security-Tools Web-App (v2)

Security Tools is a full-stack cybersecurity web application developed using React.js, Express.js, and MongoDB (used for user authentication and login management). The application offers practical security utilities including Password Strength Checker analysis with breach detection using the Have I Been Pwned API, File Integrity Validation through cryptographic hashing (MD5, SHA-256, and others), a Phishing URL and Website Checker based on domain analysis with AI-powered recommendations, and a Port Scanner that identifies open ports and their associated services.
